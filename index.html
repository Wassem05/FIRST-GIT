Hvordan bruke Git

Denne siden er laget av Wassem for å vise hva jeg har lært om Git og GitHub.

1. Starte Git

Jeg brukte git init i VS Code for å lage et lokalt repository.

2. Legge til og lagre

Jeg skrev git add . og git commit -m "første commit" for å lagre endringene mine.

3. Koble til GitHub

Jeg laget et repository på GitHub og brukte git push for å laste opp prosjektet mitt.

4. Lage branch

Jeg laget en ny branch med git checkout -b ny-branch for å jobbe videre.

5. Slå sammen

Til slutt brukte jeg git merge for å slå sammen branchene.

Jeg lærte mye om hvordan Git fungerer og hvordan man kan samarbeide i prosjekter.
